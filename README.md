
This driver is modification for the Adafruit BNO055 Breakout, and is based on Adafruit's Unified Sensor Library (Adafruit_Sensor).


## What is added? ##

Missing functions for interrupt handling

1.) setAnyMotionInterrupt

2.) resetInterrupt
